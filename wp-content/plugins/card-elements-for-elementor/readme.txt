== Description ==

Card Elements is an elementor add-on to showcase your team profiles (our team), testimonials and WordPress posts with card style/design. This is an simple and flexible way to add new elements/widgets to Elementor Page Builder.

[View Demo](https://demo.techeshta.com/elementor/card-elements/)

### 3 Ready Made Card Elements

We have designed 3 of the most useful elements to showcase/decorate your website with card styling. It allows you to climb the top of your design capabilities.

1. [Profile Card](https://demo.techeshta.com/elementor/card-elements/profile-card/) - Showcase your team members in 5+ unique styles
2. [Testimonial Card](https://demo.techeshta.com/elementor/card-elements/testimonial-card/) - Showcase your client testimonials in 2+ unique styles
3. [Post Card](https://demo.techeshta.com/elementor/card-elements/post-card/) - Showcase your posts in 2+ unique card styles
4. Many more coming soon ...

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==
= Can I add multiple Card elements with single page? =
Yes, you can. Card elements for elementor plugin gives you the ability to add **multiple card styles on same page**. You just need to drag and drop that module multiple times and select different card styles whatever you want.

=  Can I change color or customize card style as per my wish? =
Yes, You can customize it. We provided all possible options in backend to update it from default card elements.
